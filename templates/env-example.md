# .env.example Template — Per Service

Use this to generate the project's `.env.example` based on selected services.

## n8n

```
# n8n instance URL (if connecting externally)
N8N_BASE_URL=
N8N_API_KEY=
```

## GoHighLevel (GHL)

```
# GHL API credentials
GHL_API_KEY=
GHL_LOCATION_ID=
GHL_PIPELINE_ID=
GHL_CALENDAR_ID=
```

## OpenAI

```
# OpenAI API
OPENAI_API_KEY=
OPENAI_MODEL=gpt-4o
```

## Google Gemini

```
# Gemini API
GEMINI_API_KEY=
GEMINI_MODEL=gemini-2.5-pro
```

## Shopify

```
# Shopify Admin API
SHOPIFY_STORE=
SHOPIFY_ACCESS_TOKEN=
SHOPIFY_WEBHOOK_SECRET=
```

## GCP

```
# Google Cloud Platform
GCP_PROJECT_ID=
GCP_REGION=europe-west2
```

## General

```
# Application
PORT=8080
NODE_ENV=production
LOG_LEVEL=info
```
